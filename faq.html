💳 報名與點數

關於課程報名流程、點數購買與使用的相關問題

購買點數有以下幾種方式：

  1. 線上購買：登入學員儀表板，點選「購買點數」，可使用信用卡或銀行轉帳付款
  2. 課程報名時購買：在報名課程時，系統會提醒點數不足並引導至購買頁面
  3. 電話購買：撥打客服專線 0800-123-456，由專人協助處理

注意事項：

  • 點數購買後立即生效
  • 點數有效期限為購買日起一年
  • 可購買點數：最少1點，最多一次購買100點
  • 每點約等於 NT$100（實際價格依方案而定）

可以的！您可以在規定期限內取消報名：

退費標準

取消時間 退費比例 點數處理
課程開始前7天以上 100% 全數退還
課程開始前3-7天 80% 按比例退還
課程開始前1-3天 50% 按比例退還
課程當天或之後 0% 不予退還

取消方式：

  1. 登入學員儀表板，在「即將上課」中點選「取消報名」
  2. 於課程詳情頁面點選「線上取消報名」
  3. 致電客服專線由專人協助處理

當您的點數不足時，有以下解決方案：

  1. 立即購買：系統會自動提醒需要的點數數量，點選「購買點數」立即加值
  2. 合作單位加值：如果您屬於合作機構，可請單位承辦人協助加值
  3. 朋友分享：其他學員可透過「點數分享」功能贈送點數給您
  4. 課程優惠：關注官網公告，不定期有課程優惠或點數加碼活動
小提醒：建議您在點數餘額低於20點時及時補充，以免影響課程報名。系統會在點數不足時發送提醒通知。

是的！您可以將點數分享給其他註冊學員：

分享步驟：

  1. 登入學員儀表板
  2. 在「點數與分享」區塊輸入受贈者手機號碼
  3. 輸入要分享的點數數量
  4. 確認受贈者資訊無誤後點選「確認分享」
  5. 系統會發送確認Email給雙方

注意事項：

  • 每次最少分享1點，最多50點
  • 每月分享上限為100點
  • 受贈者須為已註冊會員
  • 分享後無法取消，請確認資訊無誤
  • 分享的點數有效期限與原點數相同

📜 長照積分與時數

關於長期照顧積分認證與學習時數的相關規定

具有長照積分的課程在課程列表和詳情頁面會明確標示「長照積分」。

積分課程特色：

  • 符合衛福部長期照顧專業人員訓練認證規定
  • 可計入長照專業人員繼續教育時數
  • 課程內容經相關主管機關審核認可
  • 結業後核發正式積分證明

積分申請注意事項：

  1. 需完整出席課程（出席率達80%以上）
  2. 通過課程測驗（成績70分以上）
  3. 繳交個人長照小卡影本
  4. 積分認證以實際申請結果為準
提醒：積分課程開課前會再次確認積分認證狀況，如有異動會提前通知學員。

您可以透過以下方式查詢年度學習時數：

線上查詢：

  1. 登入學員儀表板
  2. 首頁即可看到「本年學習時數」統計卡片
  3. 點選「學習歷程」可查看詳細時數記錄
  4. 可按月份、課程類別篩選查看

時數計算方式：

  • 以實際出席並完成的課程時數計算
  • 未完成的課程不計入時數
  • 線上課程以實際觀看並通過測驗的時數計算
  • 同一課程重複上課，時數不重複計算

證明申請：

  • 可於學員中心下載「學習歷程證明」
  • 提供Word和PDF兩種格式
  • 包含詳細的課程名稱、時數、日期等資訊
  • 適用於繼續教育時數認證申請

長照積分申請由本校代為辦理，您只需要：

課程前準備：

  1. 確認已上傳長照小卡至個人資料
  2. 核對個人基本資料正確性
  3. 確認出席課程

課程期間：

  1. 準時到課並完成簽到簽退
  2. 積極參與課程活動
  3. 完成課程測驗
  4. 填寫課程滿意度問卷

課程後處理：

  1. 本校彙整學員資料並向主管機關申請
  2. 約2-4週完成積分登錄作業
  3. 積分登錄完成後將發送通知Email
  4. 可於衛福部長照專業人員管理系統查詢
優勢：本校為衛福部認可的長照訓練機構，積分申請流程完整，無需學員另外處理申請手續。

💻 線上教學

關於線上直播課程與數位學習的相關問題

是的！我們提供高品質的線上直播課程。

直播課程特色：

  • 即時互動：可與講師及其他學員即時交流
  • 高畫質串流：提供Full HD畫質的課程體驗
  • 多裝置支援：電腦、平板、手機皆可觀看
  • 雲端錄影：課程結束後提供回看服務

直播前準備：

  1. 設備檢測：課前30分鐘須完成設備檢測
  2. 網路環境：建議使用有線網路或穩定的WiFi
  3. 瀏覽器：建議使用Chrome、Firefox或Safari最新版本
  4. 音訊設備：準備耳機或喇叭以獲得最佳音質

參與規範：

  • 準時進入直播間（開課前15分鐘開放入場）
  • 遵守課堂秩序，避免干擾其他學員
  • 積極參與互動討論
  • 如有問題可使用聊天室功能提問

課程錄影影片的觀看期限如下：

影片觀看期限

課程類型 觀看期限 下載權限
一般課程 課程結束後3個月 不提供
證照課程 課程結束後6個月 不提供
AIO專業課程 課程結束後1年 重要講義可下載

重看功能特色：

  • 支援倍速播放（0.5x～2x）
  • 可設定書籤標記重點內容
  • 支援字幕顯示（部分課程）
  • 觀看記錄同步雲端保存

注意事項：

  • 影片僅供個人學習使用，禁止錄製或分享
  • 觀看期限到期後影片將自動移除
  • 如需延長觀看期限，可聯繫客服協助
  • 實際規則以課程公告為準

線上課程採用數位簽到系統：

簽到流程：

  1. 進入直播間：開課前15分鐘開放，請準時進入
  2. 身份驗證：系統會自動驗證您的學員身份
  3. 簽到確認：課程開始後5分鐘內完成簽到
  4. 中場簽到：課程中段會有隨機簽到驗證
  5. 簽退確認：課程結束前完成簽退

簽到方式：

  • 自動簽到：進入直播間後系統自動記錄
  • 互動簽到：透過聊天室回應指定訊息
  • 問卷簽到：填寫課間小問卷
  • 舉手功能：使用平台舉手功能回應

出席率計算：

  • 全程參與：出席率100%
  • 遲到15分鐘內：出席率90%
  • 早退15分鐘內：出席率90%
  • 中途離線超過30分鐘：出席率將依實際參與時間計算
重要：為確保獲得長照積分，線上課程出席率需達80%以上。建議全程參與以獲得最佳學習效果。

💻 網站使用問題

關於網站使用、帳號管理與操作問題的解決方法

您可以透過以下步驟重設密碼：

線上重設：

  1. 前往登入頁面
  2. 點選「忘記密碼？」
  3. 輸入註冊時的Email或手機號碼
  4. 檢查Email信箱（包含垃圾郵件資料夾）
  5. 點選重設密碼連結
  6. 設定新密碼並確認

密碼設定規則：

  • 長度至少8個字元
  • 須包含英文字母和數字
  • 建議加入特殊符號增加安全性
  • 不可使用連續或重複的簡單數字

其他解決方案：

  • 如果沒有收到重設密碼郵件，請聯繫客服
  • 如果忘記註冊Email，可提供身份證字號和姓名給客服協助
  • 建議使用密碼管理器避免忘記密碼

遇到網站操作問題時，請依下列步驟處理：

基礎排除：

  1. 重新整理頁面（F5或Ctrl+R）
  2. 清除瀏覽器快取和Cookie
  3. 嘗試使用不同瀏覽器
  4. 檢查網路連線是否正常

建議瀏覽器設定：

  • 推薦瀏覽器：Chrome、Firefox、Safari、Edge（最新版本）
  • JavaScript：請確認已啟用
  • Cookie：請允許本站使用Cookie
  • 廣告阻擋：請將本站加入白名單

常見問題處理：

  • 頁面載入緩慢：可能是網路問題，請稍後再試
  • 按鈕無反應：檢查是否啟用JavaScript
  • 檔案無法下載：檢查瀏覽器下載設定
  • 影片無法播放：更新瀏覽器或嘗試其他瀏覽器
